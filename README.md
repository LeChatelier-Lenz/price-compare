# Web项目开发

## 技术栈
- 前端: React + mui 
- 后端: Go + Gin + Gorm
- 数据库: MySQL
- 部署: Docker + Nginx

## 项目结构
```
.
├── README.md
├── backend
│  ...
└── frontend
   ...
```

## 项目启动
- 后端: 
  ```bash
  cd backend
  go run main.go
  ```
- 前端: 
  ```bash
    cd frontend/price-compare
    npm start
    ```
- 访问: http://localhost:3000
